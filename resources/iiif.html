IIIF Endpoint

Registration API

Endpoint for institutions to register their content and make it searchable

  • POST iiif/register
  • application/json
  • HTTP/1.1

Request Fields

Name Data Type Description
institution String Institution that holds content
iiifEndpoint String IIIF endpoint of institution
content Array Information per image
- project String project associated with URL
- image String image id (for IIIF url)
- meta String Metadata of image

Example Request

        
        {
            "institution":"DaSCH",
            "iiifEndpoint":"iiif.dasch.swiss/",
            "content":[
                {
                    "project": "sweee",
                    "image": "sdfr",
                    "meta": "sdgsdfds"
                },
                {
                    "project": "scheeeee",
                    "image": "sdfs",
                    "meta": "jahui"
                }
            ]
        }

Example Responses

Successful Registration

        
        {
            status : 200,
            institution: "DaSCH",
            numberOfResources: "2", 
            requestID: 1
        }

Missing values, empty fields

        
        {
            status : 400, 
            errorMessage : [
                “Missing value for institution”,
                "empty value for iiifEndpoint",
                "content must be nonempty array"
            ]
        }
